About This Project

My favorite part of being a software engineer is being able to blend logic with design. I created this "Sketch-A-Etch" because it's a perfect representation of development: Creative Functionality.

Code allows me to be both logical and artistic, to be able to use both the right and left sides of my brain, and to convert a blank webpage, into a beautiful work of art.
